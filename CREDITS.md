## Third-Party Libraries

Planet Engine is built on the shoulders of these excellent open-source projects:

| Library | Version | License | Purpose |
|---------|---------|---------|---------|
| [Lua](https://www.lua.org/) | 5.4.7 | MIT | Embedded scripting language |
| [GLFW](https://www.glfw.org/) | 3.x | zlib/libpng | Window creation and input handling |
| [OpenGL](https://www.opengl.org/) | 4.1+ | Khronos | 3D/2D graphics rendering |
| [GLAD](https://glad.dav1d.de/) | - | MIT/Public Domain | OpenGL function loader |
| [GLM](https://github.com/g-truc/glm) | - | MIT / Happy Bunny | OpenGL Mathematics library |
| [OpenAL](https://www.openal.org/) | - | LGPL | 3D audio playback |
| [stb_image](https://github.com/nothings/stb) | - | MIT / Public Domain | Image file loading |
| [FMOD](https://www.fmod.com/) | - | Proprietary | Advanced audio engine (optional) |
